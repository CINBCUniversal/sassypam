Filter Bar

Filter bars take up full width with a filter icon to the right. Filter icons are absolutely positioned, it might look a little low here, but it perks up in the app.

<div class="filter-bar flex-center cf width-100">
    <div class="col-xs-12">
      <input class="input-search width-100 h5 em-low" placeholder="Filter for stuff!" type="text" value="">
      <div class="filter-icon pos-abs" name="button" type="submit">
        <i class="sms-glyph-filter h5 mrs"></i>
      </div>
    </div>
  </div>

Search bar

Search bars take up full width with a search icon to the right and a clear search button if needed. Usually put inside of a filter module.

<div class="search-bar width-100 cf flex-center pal">
    <div class="col-xs-10">
      <input class="input-search width-100 h5 em-low" placeholder="Search for stuff!" type="text" value="">
      <button class="search-button pos-abs" name="button" type="submit">
        <i class="sms-glyph-icon_search h5 mrs"></i>
      </button>
    </div>
    <div class="col-xs-2">
      <button class="button-clear-search em-alt h6" name="button" type="submit">Clear Search</button>
    </div>
  </div>

Sticky Panel

Sticky panels will require adding an include statement to the sticky panel mixin with color of choice.

You will also need to include whatever javascript to get the panel to fix. See the Quintiles or Finance Model pages for examples.
