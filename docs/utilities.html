Icon Wrappers

Sometimes an icon from the collection may be a different size after user interaction, such as an arrow or a plus or minus. This may cause the layout to pop or flash to adapt to the new icon size after clicking. To fix this we created several icon-wrap helper classes to be applied only to the i element that contains the icon.


<i class='h6 sms-glyph-arrow_triangle-right icon-wrap-sm' style="background-color:#F8E71C"></i>
<br>
<i class='h6 sms-glyph-arrow_triangle-down icon-wrap-sm' style="background-color:#F8E71C"></i>
