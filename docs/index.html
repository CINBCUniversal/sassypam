Side Navigation

Side Navigation (inside content area) STYLE MISSING
Sometimes you may need side navigation elements that help users navigate within a specific content area. This side navigation would typically coincide with a use of fixed or absolute positioning. We do not have a fixed module as that would be dependent on how the module is layed out within a specific application's content.
