Buttons

Buttons that live on white backgrounds should have a light grey border, and always include an inline icon that reinforces the action described in the label.

<button class="button"><i class="sms-glyph-icon_plus"></i>Add Deal</button>

Buttons that live on the blue Content Action Bar should never have icons, and feature both a transparent border and a colored background (the color corresponding to the "weight" of the action within the given screen)

<button class="button-success button">Create Deal</button>

<button class="button-primary button-sm">Create Deal</button>

</div>

Tables

Simple Table

  1. Text is aligned left, while numbers are aligned right. Negative numbers are red for emphasis.
  2. Table headers have bold font aligned with the column data.
  3. Row titles have dark grey backgrounds with a box-shadow inset.
  4. Row summaries have a blue background with the opacity turned down.
  5. Selectable rows highlight yellow.
  6. Sub headers are defined by text-indentation in the row title.
