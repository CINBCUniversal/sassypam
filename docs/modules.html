Content Title

The main page content title goes here with any additional details. You won't need the border classes if it's wrapped in a content-border class

Bravo

Gross Cash Profit
<div class="content-title-finance border-right border-left">
    <h1>Bravo</h1>
    <h6>Gross Cash Profit</h6>
  </div>

Filter Module

All filters that belong to the page should go inside of a filter module

View Report By:
  • Registrations
  • Pacing
Choose Quarter:
  • Q1
  • Q2
  • Q3
  • Q4 
<div class="filter-module">
  <div class="filter-module-section">
    <h6 class="mbm em-alt">View Report By:</h6>
    <ul class="button-group">
      <li>
        <a class="filter-button active" href="#">Registrations</a>
      </li>
      <li>
        <a class="filter-button" href="#">Pacing</a>
      </li>
    </ul>
  </div>
  <div class="filter-module-section">
    <h6 class="mbm em-alt">Choose Quarter:</h6>
    <ul class="button-group">
      <li>
        <a class="filter-button active" href="#">Q1</a>
      </li>
      <li>
        <a class="filter-button" href="#">Q2</a>
      </li>

        <a class="filter-button" href="#">Q3</a>

      <li>

      </li>
    </ul>
  </div>
</div>

Modals

Modals are modules layed on top of the page to allow the user to walk thru a specific action or alert them of something.

Modals are used with Bootstraps Modal JavaScript, please refer to Bootstrap on how to use

Alert

Quarterly percentages do not add up to 100%, please update

<div class="modal-dialog">
  <div class="modal-content">
    <div class="modal-header pal">
      <h2 class="modal-title">Alert</h2>
    </div>
    <div class="modal-body pal">
      <p>Quarterly percentages do not add up to 100%, please update</p>
    </div>
    <div class="modal-footer pal">
      <button class="button" data-dismiss="modal" type="button">Close</button>
    </div>
  </div>
</div>
