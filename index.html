Search bar

If you are using Rails, you can generate pagination automagically by using the Kaminari gem. Just use '.paginate-container'
 Otherwise, you can use the markup below!

    <div class="search-bar">
      <input class="input-search" placeholder="Search for stuff!" type="text" value="">
      <button class="button-search" name="button" type="submit"><i class="sms-glyph-icon_search prm"></i>Search</button>
      <button class="button-clear-search" data-behavior="clear_deal_search" name="button" type="submit">Clear Search</button>
    </div>

Side Navigation

Side Navigation (inside content area) STYLE MISSING
Sometimes you may need side navigation elements that help users navigate within a specific content area. This side navigation would typically coincide with a use of fixed or absolute positioning. We do not have a fixed module as that would be dependent on how the module is layed out within a specific application's content.
